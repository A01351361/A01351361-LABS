¿Cuál es la diferencia entre Internet y la World Wide Web?
  • El internet es considerado como una red de computadoras conectadas entre si alrededor del mundo, mientras que la web es considerada como una colección de páginas dentro de la red de computadoras
¿Cuáles son las partes de un| URL?
    1. Protocolo HTTP: Transferir información entre diferentes actores en la web
    2. Subdominio: Lo que aparece en el protocolo (www / es)
    3. Dominio: Nombre unico para identificar una página web
    4. Ruta: Indica páginas en un sitio web
    5. Parametro: Indican diferentes cosas, como busquedas del sitio, campañas, etc.
    6. Etiqueta: Dirige a la persona a una parte exacta de la página
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
  • GET: Solicita representación de un recurso en especifico, recupera datos
  • HEAD: Pide una respuesta similar a la peticion del GET, pero sin el cuerpo de la respuesta
  • POST: Envia una entidad a un recurso en especifico
  • PUT: Reemplaza las representaciónes del recurso de destino con la petición
  • PATCH: Aplicar modificaciones de manera parcial a un recurso
  • DELETE: Borra un recurso en especifico
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?
  • El codigo de estado de una respuesta HTTP con codigo 200 significa que el request es correcto, es una respuesta estandar para las respuestas correctas.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
  • En el HTML5 se simplifica el DOCTYPE, incluyen elementos de SVG y MathML, nuevos elementos como section, article, header, footer, entre otros. Mejora de las API. Mientras que en HTML4 todo era más dificil y complejo a comparación del HTML5.
¿Qué componentes de estructura y estilo tiene una tabla?
  • table: datos con más de una dimensión
  • caption: titulo de una tabla
  • colgroup: conjunto de una o más columnas de una tabla
  • col: Representa una columna de una tabla
  • tbody: Bloque de filas que describen datos concretos en la tabla
  • thead: etiquetas de columna en una tabla
  • tfoot: resumenes de columna en una tabla
  • tr: fila de celdas en una tabla
  • td: celda de datos en una tabla
  • th: celda encabezado en una tabla
¿Cuáles son los principales controles de una forma HTML5?
  • form: Representa un formulario
  • fieldset: Conjutno de controles
  • label: Titulo de un control de formulario
  • legend: representa tituli de fieldset
  • input: Campo de datos escrito que permite al usuario editar los datos
  • button: Representa un botón
  • select: Control que permite la selección entre un conjunto de opciones
  • textarea: Representa un control de edición de texto multilínea
  • keygen: Representa un control de par generador de llaves
  • output: Representa el resultado de un calculo
